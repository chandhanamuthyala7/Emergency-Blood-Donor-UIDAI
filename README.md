# Emergency Blood Donor UIDAI

## Project Description
Emergency Blood Donor UIDAI is a system designed to help people find blood donors quickly during emergency situations. 
The project uses a data-driven approach inspired by UIDAI Aadhaar identification concepts to organize donor information.

## Features
- Quick search for blood donors
- Organized donor data
- Simple interface
- Useful during emergency situations

## Technologies Used
- Python
- VS Code
- HTML / CSS (if used)

## Future Scope
- Integration with hospital databases
- Real-time donor location tracking
- Mobile application support
